# Адаптивный веб-сайт с динамической сменой тем и визуальными эффектами
## Описание проекта
Проект представляет собой адаптивный веб-сайт с возможностью переключения между светлой и тёмной темами. Реализованы дополнительные визуальные эффекты и анимации для улучшения пользовательского опыта.

## Основные особенности
  * Система тем с переключением по клику
  * CSS-переменные для управления темами
  * Визуальные эффекты:
    * Анимации кнопок
    * Анимированные иконки
    * Фильтры для изображений
  * Адаптивный дизайн

## Структура проекта
В проекте реализованы следующие компоненты:
  * HTML-разметка с семантической структурой
  * CSS-стили с разделением на тематические файлы
  * Медиаресурсы (изображения, шрифты)

## Технические детали
  * Адаптивность реализована через медиазапросы
  * Темы управляются через CSS-переменные
  * Переключение реализовано через JavaScript-скрипт
  * Кроссбраузерная совместимость
  * Правила анимаций и переходов
